How to use:
Start OBS → point to rtmp://localhost:1935/live with stream key local-dev-key-streamer-01
Start rtmp-ingest + transcoding-service, then press one of the quick-load buttons above.

Stream URL via nginx proxy (CORS safe):
http://localhost:8080/minio/hls-segments/live/local-dev-key-streamer-01/master.m3u8

Or direct MinIO (no CORS):
http://localhost:9100/hls-segments/live/local-dev-key-streamer-01/master.m3u8

Force a specific quality track:
http://localhost:8080/minio/hls-segments/live/local-dev-key-streamer-01/720p/index.m3u8

Note: The player uses hls.js — works in Chrome, Firefox, Edge, and Safari 14+. Safari 13 and below support HLS natively via the <video> element.
